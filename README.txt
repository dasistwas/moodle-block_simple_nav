Simple Navigation Block
----------------

This block provides a navigation as it is often used on standard websites. With this block all courses with guest access can be accessed similar to standard CMS systems.

The block is far less complex than the standard Moodle navigation block. It provides links to courses, categories and modules according to the permissions of the user 

This blocks is best for you, if you want to use Moodle as LMS and as normal website too.
